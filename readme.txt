Easy way to show a totally customizable popup to user or customer. Inside your popup, you can add shortcode, text and images thanks to WYSIWYG editor.

== Description ==

Install plugin and create your simple alert popup box.
You can using a WordPress editor to write  or insert a media in alert popup box.
There are different option you can chose:

* Show popup at the first visit of the day 
* Test mode
* Contact Form 7 ready
* Shortcode ready (you can use any shortcode)
* Full control on popup content
* Standard WordPress content editor
* Set a delay to start popup box
* Responsive mobile friendly popup
* Insert image in popup box
* Choose full width layer background color
* Choose full width layer opacity
* Set popup background color
* Set popup border width and color
* Set popup content padding

== Installation ==

1. Upload the entire 'ns-wordpress-custom-alert-popup-box' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Settings' -> 'NS WordPress Custom Alert Popup Box'
